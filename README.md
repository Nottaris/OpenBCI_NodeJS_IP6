# OpenBCI NodeJS IP6
Brain Computer Interface to control a web musicplayer

## Installation
``` npm install ``` 

## Start Musicplayer
``` npm run player ``` 

## EEG Controls

#### Start blink control
``` npm run blink ``` 

###### Start blink from file
``` npm run blinkfile ```

#### Start P300 control - beta
``` npm run p300 ```

#### Start mind control - beta
``` npm run mind ``` 


### Helpers
#### Save Data
``` npm run save ``` 
#### Plot Data
``` npm run plot ```
